# Smolsmith

Lightweight counterpart to Fablesmith. Same voice fidelity, single markdown file output.

## Commands

Load the matching file from `commands/` and follow its instructions.

## Mandates

1. **Voice first** — calibrate the author's specific techniques before writing. Never generic.
2. **Parallel execution** — use parallel agents for long-form works.
3. **The imitation test** — if a reader can't guess the author, recalibrate.
