# Smolweave

Lightweight counterpart to ReadWeave. Same analytical rigor, single markdown file output.

## Commands

Load the matching file from `commands/` and follow its instructions.

## Mandates

1. **Parallel extraction** — use parallel agents for simultaneous reading.
2. **High-fidelity excerpts** — the author's actual words, never paraphrase.
3. **Analytical annotations** — connect, contextualize, surface implications. Never restate.
